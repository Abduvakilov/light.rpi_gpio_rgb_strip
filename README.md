# RGB strip component for Home Assistant
This Home Assistant custom component lets you control RGB strip with your Raspberry Pi GPIO.

### configuration.yaml
``` yaml
 light:
   - platform: rpi_gpio_rgb_strip
    dat: 20
    clk: 21
 ```
